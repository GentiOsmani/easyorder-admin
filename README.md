# EasyOrder Admin Panel

This repository contains the Windows installer for the EasyOrder Admin Panel.

## Download

- `releases/EasyOrder Admin Panel Setup 1.0.0.exe`

## Notes

- Admin app accepts only `admin` role login.
- Staff accounts (`bartender`/`manager`) are blocked in admin mode.
- Staff terminal lock logic is implemented in the main EasyOrder codebase.
